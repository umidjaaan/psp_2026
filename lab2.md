## 3. Задание

### 3.1. Основное задание

Разработать JavaScript-логику для управления калькулятором расчета комплектации спутника.

Необходимо реализовать:
- получение элементов из DOM-дерева;
- обработку пользовательских действий (нажатий кнопок);
- выполнение базовых арифметических операций;
- вывод результата вычислений в поле `display`.

---

### 3.2. Дополнительное задание

Расширить функциональность с учетом инженерной специфики проекта:

- **Инженерные операции:** вычисление процентов, извлечение квадратного корня, расчет факториала, возведение в степень.
- **Память:** реализация кнопок работы с памятью `M+`, `M-`, `MC`.
- **UI:** переключение визуальных тем (Космическая темная / Светлая).
- **Конвертация:** перевод рублевой стоимости компонентов в доллары ($).
- **UX:** добавление кнопки быстрого ввода `000` и удаление последнего введенного символа (Backspace).

## 4. Теория и реализация (Примеры из кода)

### 4.1. Доступ к DOM и обработка событий

**Теория:** Для взаимодействия с интерфейсом используется механизм прослушивания событий `addEventListener`. Это позволяет отделить бизнес-логику калькулятора от HTML-разметки. Инициализация скрипта выполняется после того, как DOM-дерево полностью построено.

**Пример из кода:**

```javascript
document.addEventListener('DOMContentLoaded', () => {
    const displayElement = document.getElementById('display');
    const digitButtons = document.querySelectorAll('.btn-calc:not(.operator):not(.action)');

    digitButtons.forEach(button => {
        button.addEventListener('click', () => {
            const val = button.textContent === '000' ? '000' : button.textContent;
            handleDigitInput(val);
        });
    });
});
4.2. Логика инженерных вычислений
Для избежания дублирования кода применяется функция высшего порядка applyUnary. Она принимает математическую операцию в качестве аргумента и применяет её к текущему числу на экране.

Пример из кода:

JavaScript


const usdExchangeRate = 92.50; // Условный курс для пересчета стоимости деталей

const applyUnary = (operationFunc) => {
    if (!selectedOperation) {
        firstOperand = operationFunc(parseFloat(firstOperand || 0)).toString();
        updateDisplay(firstOperand);
    } else {
        secondOperand = operationFunc(parseFloat(secondOperand || 0)).toString();
        updateDisplay(secondOperand);
    }
};

// Конвертация в доллары
document.getElementById('btn_usd')
    .addEventListener('click', () => applyUnary(x => x / usdExchangeRate));

// Извлечение корня
document.getElementById('btn_sqrt')
    .addEventListener('click', () => applyUnary(x => Math.sqrt(x)));
4.3. Управление состоянием интерфейса
Изменение внешнего вида реализуется через управление CSS-классами напрямую из JavaScript. Скрипт отвечает только за переключение состояний (добавление/удаление классов), а сами визуальные правила описаны в CSS.

Пример из кода:

JavaScript


const themeToggleBtn = document.getElementById('themeToggle');

if (themeToggleBtn) {
    themeToggleBtn.addEventListener('click', () => {
        // Переключаем класс светлой темы на body
        document.body.classList.toggle('light-theme');

        // Меняем текст кнопки в зависимости от текущей темы
        if (document.body.classList.contains('light-theme')) {
            themeToggleBtn.textContent = 'Темная тема';
        } else {
            themeToggleBtn.textContent = 'Светлая тема';
        }
    });
}
4.4. Динамическое масштабирование и обработка исключений
Для корректного отображения длинных чисел в узком окне калькулятора реализована адаптация размера шрифта. Также предусмотрена защита от переполнения экрана.

Пример из кода:

JavaScript


function adjustDisplayFontSize(value) {
    const displayElement = document.getElementById('display');
    const length = value.length;

    if (length > 9 && length <= 13) {
        displayElement.style.fontSize = "1.5rem";
    } else if (length > 13 && length <= 25) {
        displayElement.style.fontSize = "1.1rem";
    } else if (length > 25) {
        displayElement.textContent = "Overflow";
        displayElement.style.color = "red";
    } else {
        displayElement.style.fontSize = "2rem"; // Дефолтный размер
        displayElement.style.color = "";
    }
}
