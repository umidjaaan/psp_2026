# CubeSat Enterprise | Лабораторные работы ИУ5

Интерактивное веб-приложение для конфигурации наноспутников формата CubeSat. Проект реализован в рамках учебного курса по разработке программного обеспечения.

## 🚀 Основной функционал
- **SPA-архитектура:** Навигация между разделами без перезагрузки страниц.
- **Интерактивный каталог:** Динамическая генерация карточек компонентов на основе данных.
- **Персонализация:** Возможность изменения стоимости компонентов с сохранением через `localStorage`.
- **3D-визуализация:** Интеграция 3D-моделей узлов спутника для детального просмотра.

## 🛠 Технологии
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.
- **Логика данных:** Web Storage API (localStorage) для управления состоянием цен.

## ⚙️ Как запустить проект
1. Клонируйте репозиторий: `git clone https://github.com/umidjaaan/psp_2026.git`
2. Перейдите в папку проекта.
3. Откройте `frontend/index.html` в браузере или используйте Live Server в VS Code.
